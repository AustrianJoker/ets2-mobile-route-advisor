About

If you've played Euro Truck Simulator 2 (ETS2) before, you are probably aware of the "Route Advisor", the little dashboard that appears on the left or right-hand side of the screen that shows you your GPS, along with cargo and truck information. The Mobile Route Advisor is a skin for Funbit's ETS2 telemetry web server, which will allow you to view most of this information on any device that has a web browser installed.

Currently, there is no GPS feature included in the Mobile Route Advisor, as the required information is not available through SCS Software's Telemetry SDK at this time.

Features

The following features are currently supported:

  • Current Speed (kmh or mph)
  • Speed Limit (coming soon in 1.1.0)
  • Remaining Distance (coming soon in 1.1.0)
  • Current gear
  • Remaining fuel
  • In-Game Time
  • Cargo Name and Weight (tons or kilograms)
  • Source
  • Destination
  • Deadline
  • Reward / Job Income
  • Time Remaining
  • Truck Damage
  • Trailer Damage

Additionally, various configuration options are available to display relevant information to the user how they want it to be displayed:

  • Language (currently 6 languages are supported)
  • Time Format (12-hour vs 24-hour)
  • Speed Units (kmh vs mph)
  • Weight Units (kilograms vs tons)
  • Currenty Code (EUR, CHF, CZK, GBP, PLN, HUF)
