About

If you've played Euro Truck Simulator 2 (ETS2) before, you are probably aware of the "Route Advisor", the little dashboard that appears on the left or right-hand side of the screen that shows you your GPS, along with cargo and truck information. The Mobile Route Advisor is a skin for Funbit's ETS2 telemetry web server, which will allow you to view most of this information on any device that has a web browser installed.

Currently, there is no GPS feature included in the Mobile Route Advisor, as the required information is not available through SCS Software's Telemetry SDK at this time.
